## APP-45 Customer home screen wireframe

### Layout (mobile-first)
```
[Top bar]
  - Logo
  - Location / pickup area selector (optional)
  - Account menu (sign in)

[Search & filters]
  - Search input
  - Filter chips / dropdown:
    - Date
    - Availability
    - Distance (optional)

[Listings grid/list]
  - Card per listing:
    - Listing title
    - Restaurant name
    - Pickup window (start - end)
    - Qty available
    - CTA area:
      [View details]

[Empty state]
  - "No listings available"
  - [Try again]
```

### States
- Loading skeleton cards.
- Empty.
- Error with safe retry.
